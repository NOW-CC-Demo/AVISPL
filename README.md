# AVISPL

Project management structure for AVISPL.

## Directory Structure

- **00_Inbox/** - Incoming items and notes
  - **calls/** - Call recordings and notes (internal/external)
  - **emails/** - Email correspondence
  - **notes/** - General notes
- **01_Customers/** - Customer-specific folders
- **10_PromptTemplates/** - AI prompt templates
- **20_Demo_Library/** - Demo materials and resources
- **30_CommunicationTemplates/** - Email and communication templates
- **99_Assets/** - Project assets
  - **Project_Overview/** - Project documentation
  - **Communications/** - Communication materials
  - **POC_Documents/** - Proof of concept documents
